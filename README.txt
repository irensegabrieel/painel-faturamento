Como usar:

1. Deixe o app.py junto da pasta dashboard criada pelo extrator.
2. Abra o Prompt de Comando nessa pasta.
3. Instale as dependências uma vez:
   pip install -r requirements.txt
4. Rode:
   streamlit run app.py
5. O navegador vai abrir o painel.

Para publicar depois:
- Suba app.py, requirements.txt e a pasta dashboard para um repositório no GitHub.
- Depois conecte esse repositório no Streamlit Community Cloud.
